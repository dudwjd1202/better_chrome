# ⏰ 웹사이트 사용 시간 & 기회비용 분석기


## 📌 프로젝트 소개

**웹사이트 사용 시간 & 기회비용 분석기**는 크롬 익스텐션으로, 사용자가 방문하는 각 웹사이트에서 소비하는 시간을 자동으로 측정하고, 이를 금전적 가치로 환산하여 보여줍니다. 또한 탭 전환 횟수를 추적하여 집중도를 분석함으로써, 더 생산적인 웹 브라우징 습관을 만들 수 있도록 돕습니다.

### 💡 개발 동기

- 하루에 몇 시간씩 YouTube, SNS에서 시간을 보내는지 모르는 경우가 많습니다
- 시간의 가치를 금전적으로 환산하면 더 강력한 동기부여가 됩니다
- 잦은 탭 전환은 집중력 저하의 신호입니다

## ✨ 주요 기능

### 1️⃣ 웹사이트별 시간 측정
- 📊 각 도메인(예: youtube.com, google.com)별로 사용 시간 자동 측정
- 🎯 활성 탭만 측정하여 정확한 사용 시간 기록
- 💾 모든 데이터는 로컬에 안전하게 저장

### 2️⃣ 기회비용 계산 (Time is Money)
- 💰 사용자의 시급을 입력받아 저장
- 💸 각 웹사이트에서 소비한 시간을 금액으로 환산
- 📈 총 손실 비용을 한눈에 확인
- **공식**: `(사용 시간(초) / 3600) × 시급 = 기회비용`

### 3️⃣ 집중도 분석
- 🔄 탭 전환 횟수 실시간 추적
- ⚠️ 50회 이상 전환 시 "산만함 경고" 표시
- 📉 집중력 저하를 시각적으로 알림

### 4️⃣ 유휴 상태 감지
- 😴 60초 이상 활동이 없으면 자동으로 측정 일시정지
- 🎬 영상 시청 등은 계속 측정
- 🔋 불필요한 리소스 소비 방지

### 5️⃣ 직관적인 대시보드
- 📊 오늘의 집중도 요약 (총 시간, 총 비용, 탭 전환)
- 🎨 그라디언트 디자인으로 시각적 강조
- 🖼️ 각 사이트의 파비콘 표시
- 📱 사용 시간 기준 내림차순 정렬

## 🚀 설치 방법

### 방법 1: 소스코드로 설치 (개발자 모드)

1. **저장소 클론**
   ```bash
   git clone https://github.com/dudwjd1202/better_chrome.git
   cd time-cost-tracker
   ```

2. **크롬 익스텐션 로드**
   - 크롬 브라우저에서 `chrome://extensions/` 접속
   - 우측 상단의 **"개발자 모드"** 활성화
   - **"압축해제된 확장 프로그램을 로드합니다"** 클릭
   - 다운로드한 폴더 선택

3. **완료!** 🎉
   - 브라우저 툴바에 익스텐션 아이콘이 나타납니다

### 방법 2: Chrome 웹 스토어 (예정)
_현재 준비 중입니다_

## 📖 사용 방법

### 1단계: 시급 설정
1. 익스텐션 아이콘을 클릭하여 팝업 열기
2. 상단의 **"💰 내 시급"** 입력란에 시급 입력 (예: 15000)
3. **"저장"** 버튼 클릭

### 2단계: 자동 측정 시작
- 별도 설정 없이 자동으로 시간 측정이 시작됩니다
- 활성 탭의 도메인별로 시간이 누적됩니다

### 3단계: 통계 확인
- 언제든 익스텐션 아이콘을 클릭하여 통계 확인
- **대시보드**에서 총 사용 시간, 총 손실 비용, 탭 전환 횟수 확인
- **사이트별 상세 분석**에서 각 사이트의 시간과 비용 확인

### 4단계: 데이터 관리
- **"🗑️ 기록 삭제"** 버튼으로 모든 기록 초기화
- 시급 설정은 삭제 후에도 유지됩니다

## 💻 기술 스택

- **Manifest V3**: 최신 크롬 익스텐션 표준
- **Vanilla JavaScript**: 순수 자바스크립트로 구현
- **Chrome APIs**:
  - `chrome.tabs`: 탭 상태 감지
  - `chrome.storage.local`: 데이터 저장
  - `chrome.idle`: 유휴 상태 감지
- **CSS3**: 그라디언트 및 모던 UI 디자인

## 📊 기대 효과

### 개인적 측면
- ⏰ **시간 관리 개선**: 어디에 시간을 쓰는지 명확히 파악
- 💰 **경제적 인식**: 시간의 금전적 가치 체감
- 🎯 **집중력 향상**: 탭 전환 경고로 산만함 방지
- 📈 **생산성 증가**: 불필요한 웹서핑 감소

### 실제 사용 예시

**시급 15,000원으로 설정한 경우:**

| 웹사이트 | 사용 시간 | 기회비용 |
|---------|----------|---------|
| YouTube | 2시간 30분 | 37,500원 |
| Instagram | 1시간 20분 | 20,000원 |
| Facebook | 45분 | 11,250원 |
| **합계** | **4시간 35분** | **68,750원** |

→ 하루에 약 7만원, 한 달이면 **210만원**의 기회비용 발생! 💸

## 🔧 주요 코드 설명

### background.js - 시간 측정 로직
```javascript
// 탭 전환 시 이전 사이트 시간 저장 및 새 사이트 측정 시작
chrome.tabs.onActivated.addListener(async (activeInfo) => {
  await stopTracking();
  await incrementSwitchCount(); // 탭 전환 횟수 증가
  await updateCurrentTab();
});
```

### popup.js - 기회비용 계산
```javascript
// 비용 계산 함수
function calculateCost(seconds, hourlyWage) {
  if (!hourlyWage || hourlyWage <= 0) return 0;
  const hours = seconds / 3600;
  return Math.floor(hours * hourlyWage);
}
```
